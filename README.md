# educa. - Full-Stack Production-Grade O/L Learning Management System (SaaS)

An enterprise-level, hyper-optimized Learning Management System (LMS) designed for Grade 10 and 11 Sri Lankan national curriculum teachers and students. Built with a highly sophisticated serverless frontend architecture, robust parameterised conditional locking, and standalone data layer isolation.

🚀 Live System Link: [educa-ol-learning-platform.vercel.app](https://vercel.app)

---

## 💎 Architectural Highlights & Engineering Excellence

- Standalone Dynamic Multitenancy Framework: Managed single-container routing (/admin/:subject) utilizing advanced React Hooks (useParams) to achieve strict programmatic boundary locks between independent teaching faculties (Mathematics, Science, English).
- Encrypted Multi-role Gate Security: Implemented a unified zero-collision access gate supporting individual 4-digit tokenised parent pins, secure custom child string passwords, and global administrative credential validations with automatic session persistence handlers.
- Adaptive Responsive Layouts & Breakpoint Gates: Designed customized device-type gates restricting sensitive administrative modules exclusively to Desktop and Laptop resolutions while transforming mobile environments into interactive App-Style Navigation grids utilizing modular SASS architectures.
- Decoupled Real-time Analytical Engine: Programmed a zero-dependency local metrics calculator rendering structural parent progress receipts, custom multi-tier financial ledger data tables, and an autonomous algorithmic countdown class scheduler mimicking microservices behaviors.
- Native Low-footprint Document Automation: Engineered an analytical CSV export micro-module compiling system-state matrices directly within browser runtime memory, facilitating instant offline spreadsheet processing without inflating bundle sizes.

## 🎛️ Administrative Faculty Core Vaults

1.  ➕ Student Onboarding Vault: Features centralized direct student registration, cryptographic password allocations, and autonomous structural ID generation maps.
2.  📉 Grade & Score Analytics Vault: Powered by fuzzy matching logical algorithms tracking alphanumeric criteria (ID, Name, PIN, or mobile strings) with integrated color-coded performance indices.
3.  💰 Financial Ledger Vault (Payments): Combines an autonomous total collector compiler, structural client-side CSV matrix exporter, and localized international mobile format auto-standardizers (94xxxxxxxxx) linked directly with custom WhatsApp text string APIs.
4.  📅 Automated Absentee Vault: Features algorithmic attendance array sorting with integrated dynamic token masking capabilities to anonymize identity patterns on public parent indices (XXXX Masking Logic).
5.  📢 Live Notice Broadcast Vault: Serves as a centralized pub-sub communication layer pushing instantaneous markdown notifications to structural student dashboards and secure parent terminal points simultaneously.
6.  ⏳ Countdown Schedule Vault: Operates an adaptive calendar clock binding date strings against browser runtime parameters alongside full manual override triggers (FORCE_ACTIVE, POSTPONED, CLOSED).
7.  📁 Academic Document Core Vault: Serves clean dynamic URL allocations categorizing remote file links across localized tabs for class tutes, formula papers, and exam documents.
